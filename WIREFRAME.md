## Color Palette for Wireframe

```
Primary Colors:
- Deep Blue:      #1E3A5F    (Headers, Primary Actions)
- Electric Blue:  #3B82F6    (Buttons, Links, Highlights)
- Teal:           #14B8A6    (Success States, Verified)
- Emerald:        #10B981    (Positive Metrics)

Accent Colors:
- Amber:          #F59E0B    (Warnings, Pending)
- Rose:           #F43F5E    (Errors, Alerts, Bias Issues)
- Purple:         #8B5CF6    (AI Features, Agents)
- Cyan:           #06B6D4    (Graphs, Charts)

Neutral Colors:
- Slate Dark:     #1E293B    (Text Primary)
- Slate:          #64748B    (Text Secondary)
- Slate Light:    #94A3B8    (Text Muted)
- Gray:           #F1F5F9    (Backgrounds)
- White:          #FFFFFF    (Cards, Panels)
```

## Typography

```
Primary Font:     Inter (Headings, Body)
Monospace Font:  JetBrains Mono (Code, Technical)
Weights:          400 (Regular), 500 (Medium), 600 (Semi-Bold), 700 (Bold)
Sizes:            12px (Small), 14px (Body), 16px (Large), 20px (H3), 24px (H2), 32px (H1)
```
